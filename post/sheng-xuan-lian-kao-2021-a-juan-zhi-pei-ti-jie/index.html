[省选联考 2021 A 卷] 支配 题解

2021-04-17 6 min read # 信

一个究极暴力做法

题意简述

  • 给定一张有向图 GGG，定义一个点 iii 的支配集是所有满足如下性质的点：删除该点后 111 不再可达 iii。
  • 回答 qqq 个询问 <pj,qj><p_j,q_j><pj​,qj​>，询问的是加入边 pj→qjp_j\rightarrow q_jpj​→qj​ 后多少个点的支配集发生变化。
  • 1≤n≤30001\leq n\leq 30001≤n≤3000，1≤q≤200001\leq q\leq 200001≤q≤20000。

解法

​ 一步一步推。

​ 以下是考场上的思考：

  1. 加入一条边后一个点的支配集要么变小，要么不变。因为这条边使整张图 “ 更加联通 ”。
  2. 支配的传递关系：若 xxx 支配 yyy，yyy 支配 zzz，则 xxx 支配 zzz。因为删除 xxx 后 yyy 被割开，而 yyy 一旦不可达 zzz 也就随之不可达了。
  3. 支配的链式关系：若 xxx 支配 aaa，yyy 也支配 aaa，则 x,yx,yx,y 之间也存在支配关系。如不然则必定有一种情况，使得删除其中一个点后，111 可从另一个点到达 aaa。
  4. 支配树：对于一个点 xxx，设 DxD_xDx​ 为它的支配集，**则 DxD_xDx​ 中必定有且仅有一个点 faxfa_xfax​，使得 DxD_xDx​ 中除 faxfa_xfax​ 的点都支配 yyy。**这个性质可由上面的观察得出。我们新建一棵以 111 为根的树，其中包含所有 (x,fax)(x,fa_x)(x,fax​) 无向边，此即为该无向图的支配树。
  5. xxx 的支配集为支配树上所有 xxx 的祖先。
  6. xxx 的支配集中，faxfa_xfax​ 是支配集大小最大的那个点。这点可由前面得出。通过这点可以暴力地构建支配树。
  7. 加入一条边后，xxx 的支配集发生改变时，要么 faxfa_xfax​ 的支配集发生改变，要么 faxfa_xfax​ 不再是 xxx 的支配点。这点可由上面支配树的定义看出。
  8. 加入一条边 p→qp\rightarrow qp→q 后，faxfa_xfax​ 不再是 xxx 的支配点，当且仅当删除 faxfa_xfax​ 后 111 可达 ppp，qqq 可达 xxx。可能有一些诸如 q=faxq=fa_xq=fax​ 的边角情况，特殊判掉即可。

​ 于是出现了一个很暴力的做法。该做法分为 O(n2)O(n^2)O(n2) 的预处理和 O(nq)O(nq)O(nq) 的询问两部分。

​ 依次遍历所有点，看看把它删掉后哪些点不再可达。这样可以求出每个点的支配集。由前面的第六点思考，我们得以构建支配树。构建完后，对于每个点 xxx，我们把它的父亲删掉，看看哪些点仍从 111 可达，哪些点仍可达 xxx。暴力 BFS，复杂度 O(n2)O(n^2)O(n2)。

​ 查询时在树上 DFS。运用第七点和第八点综合判定即可。单次复杂度 O(n)O(n)O(n)。

​ 总复杂度 O(n2+nq)O(n^2+nq)O(n2+nq)。空间 O(n2)O(n^2)O(n2)。可以通过本题。

​ 下面是巨丑考场代码。有微调。

#include<bits/stdc++.h>
#define N 3005
using namespace std;
int n,m,Q,p,q,fa[N+1],fl[N+1][N+1],t[N+1],tfl[N+1][N+1],gfl[N+1];
vector<int>bi[N+1],fbi[N+1],o[N+1];
pair<int,int>q0[100001];
bool comp(int x,int y){return o[x].size()<o[y].size();}
void putin(){
	cin>>n>>m>>Q;
	for(int i=1;i<=m;i++)cin>>p>>q,bi[p].push_back(q),fbi[q].push_back(p);
	for(int i=1;i<=Q;i++)cin>>q0[i].first>>q0[i].second;
}
inline void dfs0(int x,int np){
	fl[x][np]=1;
	if(x==np)return;
	for(int i=0;i<bi[x].size();i++){
		int ni=bi[x][i];
		if(!fl[ni][np])dfs0(ni,np);
	}
}
void ycl(){
	dfs0(1,0);
	for(int i=1;i<=n;i++)t[i]=i;
	for(int i=1;i<=n;i++){
		dfs0(1,i);
		for(int j=1;j<=n;j++)if(fl[j][0]&&!fl[j][i])o[j].push_back(i);
	}
	sort(t+1,t+n+1,comp);
	for(int i=1;i<=n;i++){
		for(int j=0;j<o[i].size();j++){
			if(o[o[i][j]].size()==o[i].size()-1){
				fa[i]=o[i][j];break;
			}
		}
	}
}
inline void dfs1(int x,int np,int nr){
	if(x==nr)return;
	tfl[x][np]=1;
	for(int i=0;i<fbi[x].size();i++){
		int ni=fbi[x][i];
		if(!tfl[ni][np])dfs1(ni,np,nr);
	}
}
void cal0(){
	for(int i=2;i<=n;i++){
		dfs1(i,i,fa[i]);
	}
	for(int i=1;i<=Q;i++){
		int nans=0;
		for(int j=1;j<=n;j++)if(fl[q0[i].first][fa[j]]&&tfl[q0[i].second][j]&&fa[j]!=1&&fa[j]!=q0[i].first)gfl[j]=1;
		for(int j=1;j<=n;j++)if(gfl[fa[t[j]]])gfl[t[j]]=1;
		for(int j=1;j<=n;j++)if(gfl[j])nans++,gfl[j]=0;
		cout<<nans<<'\n';
	}
}
signed main(){
	putin();
	ycl();
	cal0();
	return 0;
}
